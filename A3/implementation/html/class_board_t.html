BoardT::BoardT ( std::vector< CardT >  cards)

Create a Forty Thieves gamebaord.

A Forty Thieves gameboard is created with a tableau of 10 cardstacks, a foundation of 8 cardstacks, a deck and an empty waste stack. It will be the default gameboard for the game.

Parameters
cardsA deck of cards containing two cards of each rank & suit is distributed randomly throughout the gameboard.

◆ is_win_state()

bool BoardT::is_win_state ( )

Will return a bool value that checks if the game is won.

The function will check if every foundation stack is non-empty and if the top card is a KING. If true, the function will return true as these are the conditins of a complete game.

Returns
A boolean value of whether the user has won the game or not.
