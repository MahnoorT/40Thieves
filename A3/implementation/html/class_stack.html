Detailed Description

template<typename Item>
class Stack< Item >

General stack ADT used to implmenent a stack of cards (CardStackT)

This abstract data type is used to construct a default stack that will be used to construct any stack of cards in the game. The ADT allowed for Items to be pushed onto the stack, popped off, access the top element, get the size and return a sequence of Items.
